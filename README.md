# Daily Calorie Tracker

### 프로젝트 소개
데일리 칼로리 트래커입니다.  
음식 이름과 탄수화물, 단백질, 지방을 입력하면 음식 카드가 생성되고, 저장된 데이터를 기준으로 총 칼로리와 영양소 차트를 원하는 버전(일간/주간/월간)으로 확인할 수 있습니다.
